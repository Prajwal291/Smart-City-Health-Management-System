# 🏥 Smart City Health Management System (SHMS)

A role-based health management platform built for Smart Cities using FastAPI and SQLite.

## 🚀 Features
- 3 Role-based Panels: Admin, Hospital, and User dashboards
- Appointment Booking: Users can book appointments with doctors
- Bed Management: Real-time hospital bed availability tracking
- Doctor Listings: Browse doctors by hospital
- SOS Alerts: Emergency alert system for hospitals
- Authentication: Secure login and registration with validation

## 🛠️ Tech Stack
- Backend: Python, FastAPI, SQLAlchemy
- Database: SQLite
- Frontend: HTML, CSS, Jinja2 Templates

## ⚙️ How to Run
pip install -r requirements.txt
python main.py

Then open: http://localhost:8000
